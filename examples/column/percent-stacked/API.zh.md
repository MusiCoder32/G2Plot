说明： **required** 标签代表生成图表的必选配置项，**optional** 标签代表生成图表的可选配置项。

### 特殊配置

#### stackField: string

**required**

数据集中的分组字段名，通过该字段的值，柱子将会被分割为多个部分，通过颜色进行区分。
